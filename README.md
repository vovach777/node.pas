# node.pas
Asynchronous Event-driven server applications framework

dep: libuv, OpenSSL 1.1

platforms: win32,win64, ?linux64

NodeJS-like ecosystem
---------------------   
* Buffer
* loop
* setImmediate
* nextTick
* EventEmitter
* setTimeout
* setInterval
* tcp
* pipe
* tty
* child process
* Http(s) server
* Promises
* JSON
* Http(s) client

There is no documentaion et. 
You can study examples for basical usage.
